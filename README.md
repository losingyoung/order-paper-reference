# order-paper-reference
论文参考文献排序
[地址](https://losingyoung.github.io/order-paper-reference/)

## 使用方法
一条参考文献以[]开头，[]里面可以放数字或空, 一条文献不要主动enter换行，超出屏幕自动换行即可
